# 小红书内容质检规范

## 一、小红书社区规范要点

### 绝对红线（违反即限流/封号）

1. **虚假宣传**
   - 不得使用"最好""第一""绝对"等绝对化用语
   - 不得虚构使用效果或数据
   - 医美/保健品必须标注"个人体验，非医疗建议"

2. **引流行为**
   - 不得出现微信号、QQ、淘宝链接
   - 不得暗示"私信下单""加好友"
   - 合法引流方式：小红书店铺、薯条投放

3. **侵权内容**
   - 不得盗用他人图片（即使 AI 生成也要标注）
   - 品牌名可以提及，但不能冒充品牌方
   - 不得抄袭其他创作者的文案

4. **敏感话题**
   - 避免涉及政治、宗教、歧视
   - 避免过度焦虑营销（容貌焦虑、身材焦虑）
   - 避免未标注的广告内容

### 灰色地带（可能被限流）

- 频繁编辑已发布的笔记
- 短时间内大量发布
- 标签和内容不相关
- 使用第三方工具刷量
- 过度使用热门标签但内容不匹配

## 二、敏感词检查清单

### 绝对化用语（广告法）
```
最好、最佳、最优、第一、顶级、极致、
100%、永远、万能、绝对、完美、
国家级、驰名、特供、独一无二
```

### 竞品引流词
```
微信、淘宝、拼多多、抖音、京东、
闲鱼、1688、代购、私聊、加我、
私信下单、V我、vx、wx、tb
```

### 医疗相关敏感词
```
治疗、治愈、药效、处方、根治、
疗效显著、药到病除、纯天然无副作用
```

### 夸大功效词
```
秒杀一切、全网最低、断货王、
不买后悔一辈子、再不买就没了
```

### 安全替代表达
| 违规表达 | 安全替代 |
|---------|---------|
| 最好用的 | 我个人觉得很好用的 |
| 全网最低价 | 我买到的最低价 |
| 一定要买 | 强烈建议试试 |
| 治好了我的痘痘 | 我的痘痘有了明显改善 |
| 100%有效 | 对我个人来说很有效 |

## 三、质量评分标准

### 合规性（25分）

| 分值 | 标准 |
|------|------|
| 25 | 无任何敏感词，内容合规 |
| 20 | 有 1 个可替换的灰色用语 |
| 15 | 有明确的夸大用语需修改 |
| 10 | 有引流嫌疑的表述 |
| 0  | 有明确违规内容 |

### 文案质量（25分）

| 分值 | 标准 |
|------|------|
| 25 | 标题吸引力强，正文结构清晰，口语化自然 |
| 20 | 标题合格，正文通顺，略有改进空间 |
| 15 | 标题普通，正文较长或结构不清 |
| 10 | 标题无吸引力，正文生硬或过短 |
| 5  | 文案质量明显不达标 |

### 平台适配度（25分）

| 分值 | 标准 |
|------|------|
| 25 | 标题字数合理，正文300-800字，标签15-20个 |
| 20 | 基本合规但某项参数偏离最佳范围 |
| 15 | 标签不足10个或正文过短/过长 |
| 10 | 多项参数不达标 |
| 5  | 完全不符合平台规范 |

### 互动潜力（25分）

| 分值 | 标准 |
|------|------|
| 25 | 有钩子、有干货、有互动引导、有情感共鸣 |
| 20 | 有3项达标 |
| 15 | 有2项达标 |
| 10 | 只有1项达标 |
| 5  | 无任何互动潜力设计 |

## 四、互动数据预估模型

### 影响因素权重

```
最终互动 = f(内容质量, 话题热度, 发布时间, 账号权重)

内容质量: 40%  ← 质检评分
话题热度: 30%  ← 选题
发布时间: 15%  ← 时机
账号权重: 15%  ← 粉丝基础
```

| 70-79   | 100-200  | 50-100   | 10-30  |
| 60-69   | 30-100   | 20-50    | 5-15   |
| <60     | <30      | <20      | <5     |

## 六、常见问题修复方案

| 问题 | 严重度 | 修复方案 |
|------|--------|---------|
| 标题无 emoji | 🔵 低 | 在开头添加1个表意 emoji |
| 正文 <300 字 | 🟡 中 | 补充使用体验/步骤细节 |
| 无互动引导 | 🟡 中 | 结尾加"你们觉得呢？" |
| 有绝对化用语 | 🔴 高 | 改为"我个人体验"口吻 |
| 有引流词 | 🔴 高 | 删除或改用平台内方式 |
| 标签 <10个 | 🟡 中 | 补充长尾+通用标签 |
| 正文太书面 | 🟡 中 | 口语化改写，加入语气词 |
