# 小红书爆款文案写作指南

## 一、标题公式库

### 5 大爆款标题公式

**1. 数字型**（点击率最高）
- `{数字}个{方法/好物}让你{效果}`
- 示例：5个方法让你的皮肤白到发光
- 要点：奇数比偶数效果好，数字越具体越好

**2. 疑问型**（引发好奇）
- `为什么{现象}？{暗示答案}`
- 示例：为什么你的穿搭总是不好看？可能忽略了这一点
- 要点：问出目标人群的痛点

**3. 对比型**（制造反差）
- `{A} vs {B}的{主题}` 或 `从{差}到{好}只用了{方法}`
- 示例：月薪3千 vs 月薪3万的护肤方式
- 要点：反差越大越吸引人

**4. 紧迫型**（制造焦虑）
- `后悔没早知道的{数量}个{主题}`
- 示例：后悔没早知道的8个厨房神器
- 要点：暗示不看会有损失

**5. 共鸣型**（情绪触发）
- `{人群}一定要看的{主题}`
- 示例：打工人一定要看的摸鱼神器
- 要点：精准圈定目标人群

### 标题 Emoji 使用规则
- 开头放 1 个表意 emoji：📝✨🔥💡🎯
- 不要超过 2 个 emoji
- emoji 要和内容相关，不要随意堆砌

## 二、正文结构模板

### 万能结构：SCQA 模型

```
S (Situation) 场景代入
  "姐妹们有没有这种经历..."

C (Complication) 痛点冲突
  "每次都XXX，简直崩溃..."

Q (Question) 问题引出
  "到底怎么才能XXX？"

A (Answer) 解决方案
  "今天分享我的独家方法..."
```

### 开头钩子（前两行，信息流可见）

必须在前两行抓住注意力：
- **痛点共鸣**："姐妹们！如果你也在为XXX发愁..."
- **数据震撼**："用了2年，试了50+款，终于找到..."
- **反常识**："千万别再XXX了！大多数人都做错了"
- **利益前置**："省了3000块，效果还更好..."
- **悬念引入**："结果连我自己都被惊到了..."

### 正文排版规则

1. **段落**：每段 2-3 行，不超过 4 行
2. **分隔符**：用 emoji 作段落分隔
   - 推荐用：✨ 💫 🌟 ⭐ 🔸 🔹 ▪️ ◽
   - 教程步骤：1️⃣ 2️⃣ 3️⃣ 4️⃣ 5️⃣
   - 清单列举：✅ ☑️ ✔️
3. **重点标记**：「关键词」加书名号突出
4. **语气词**：适度使用 "！" "..." "～" 制造情绪
5. **空行**：重要内容前后留空行

### 结尾互动引导模板

- **提问式**："你们平时都怎么XXX？评论区聊聊～"
- **投票式**："A还是B？评论区告诉我你的选择！"
- **征集式**："还有什么想看的内容？评论区留言～"
- **共鸣式**："有同款烦恼的姐妹举个手🙋‍♀️"

## 三、小红书口语化写作技巧

### 称呼词库
- 亲切：姐妹们、宝子们、集美们、家人们
- 自称：本人、本宝、老学姐、过来人
- 不要用：亲、各位网友、读者（太正式）

### 常用语气词
- 惊喜：天呐！绝了！离谱！真的会谢！
- 推荐：安利！必入！谁懂啊！
- 感叹：绝绝子、YYDS、无敌了
- 转折：但是！重点来了！
- 注意：适度使用，不要每句都有（最多 30% 的句子）

### 避免的写法
- ❌ 太书面化："经过长期实践验证..."
- ✅ 口语化："用了两个月真心觉得..."
- ❌ 太冷淡："以下是产品评测结果"
- ✅ 有温度："跟你们说这个我真的用到空瓶了"

## 四、标签策略

### 标签组合公式（15-20个）

```
核心标签 × 3     高搜索量的主题词
                  如：#护肤 #防晒 #美白

品类标签 × 3     产品/领域分类词
                  如：#面霜推荐 #平价护肤 #学生党护肤

长尾标签 × 5     具体场景/需求
                  如：#油皮防晒 #30块防晒 #通勤防晒

热门标签 × 3     平台当前热门话题
                  如：#好物推荐 #拯救废脸计划

通用标签 × 3-5   万能引流标签
                  如：#小红书 #种草 #好物分享 #真实测评
```

### 标签放置位置
- 正文末尾（最常见）
- 评论区置顶（不影响阅读体验）
- 不要放在正文中间（影响阅读）

## 五、文案检查清单

发布前逐项确认：

- [ ] 标题 8-20 字，含 emoji，有吸引力？
- [ ] 前两行是否足够抓人？
- [ ] 正文 300-800 字？
- [ ] 段落清晰，有 emoji 分隔？
- [ ] 口语化表达，不生硬？
- [ ] 有个人体验/感受？
- [ ] 结尾有互动引导？
- [ ] 标签 15-20 个，组合合理？
- [ ] 无敏感词 / 违禁词？
- [ ] 整体读起来通顺自然？
